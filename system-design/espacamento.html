Quando usar cada um

TokenUso típico
--space-1Gaps de ícone-texto, micro-espaçamento
--space-2Gap entre elementos relacionados (icon+label)
--space-3Padding interno de chips, gap em listas densas
--space-4Padding padrão de cards, gap entre componentes
--space-5Padding de cards confortável, margem entre subseções
--space-6Padding generoso, margem entre subseções
--space-8Margem entre seções de página, padding de hero
--space-10Padding do main, espaçamento de página
--space-12Margem grande entre blocos

Regras

Múltiplos de 4: qualquer valor não-token deve ser múltiplo de 4.
Não usar números mágicos: nunca padding: 13px, sempre var(--space-3).
