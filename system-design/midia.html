Grid 3 colunas exibindo todas as mídias trocadas em uma conversa. Vídeos têm overlay com play.

Regras

Lazy loading sempre: use loading="lazy" em todas as imagens da galeria/info panel.
Aspect ratio fixo: use aspect-ratio: 1 em thumbs para evitar layout shift enquanto carregam.
File thumbs são SVG inline: sem assets externos, 0 requisições, escalam perfeitamente.
